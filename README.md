# 运行须知：

前端环境搭建
a)	安装node.js
b)	安装nrm：  npm i -g nrm
c)	查看镜像源：nrm ls
d)	选定镜像源：nrm use taobao
e)	下载镜像文件：npm i
f)  安装echarts：npm install echarts --save
