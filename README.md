#### 前端环境搭建 

安装node.js 

``` bash
# 安装nrm：
npm i -g nrm

# 查看镜像源：
nrm ls

# 选定淘宝镜像源：
nrm use taobao

# 下载镜像文件：
npm i

# 安装echarts：
npm install echarts --save 

# 安装Quill-Editor：
npm install vue-quill-editor -S
```
