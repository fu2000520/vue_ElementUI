### 运行须知：
#### 前端环境搭建  
*	安装node.js  
*	安装nrm：  npm i -g nrm  
*	查看镜像源：nrm ls  
*	选定镜像源：nrm use taobao  
* 下载镜像文件：npm i  
* 安装echarts：npm install echarts --save
